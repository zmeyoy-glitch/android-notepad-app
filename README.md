# 📱 Android Notes App (Native Kotlin)

Приложение для заметок с временными слотами 17:00-20:00 (шаг 15 минут).

## ⏰ Временные слоты

| Слот | Время |
|------|-------|
| 1    | 17:00 - 17:15 |
| 2    | 17:15 - 17:30 |
| 3    | 17:30 - 17:45 |
| 4    | 17:45 - 18:00 |
| 5    | 18:00 - 18:15 |
| 6    | 18:15 - 18:30 |
| 7    | 18:30 - 18:45 |
| 8    | 18:45 - 19:00 |
| 9    | 19:00 - 19:15 |
| 10   | 19:15 - 19:30 |
| 11   | 19:30 - 19:45 |
| 12   | 19:45 - 20:00 |

## 📦 Технологии

- **Language:** Kotlin
- **UI:** Jetpack Compose (современный UI)
- **Database:** Room (для сохранения заметок)
- **Architecture:** MVVM + Clean Architecture
